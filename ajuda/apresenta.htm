.::Apresentação do Sistema

O PageMaster é um sistema para a confecção de formulários ASP, envio de e-mail atraves dos objetos AspMail® e CDONTS®.

O formulário são criados com base de dados Access® contendo módulos para consulta, cadastro, exclusão e alteração de registros. Tudo isso criado com muita facilidade, e sem saber nada sobre programação para web.

Para o envio de e-mail são criados duas páginas ASP, uma para a digitação do dados e outra para o envio dos mesmo. Facil e simples o PageMaster lhe proporciona facilidade e agilidade na confecção de páginas para melhorar a aparencia de seu site e lhe poupando muito tempo.
