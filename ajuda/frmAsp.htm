Abrindo uma base de dados Access®...

Inicie um novo projeto de formulário ASP selecionando o item com o descrito abaixo:

 

Selecione o arquivo Access® e ele será exibido na tela de abertura ao lado  com suas repectivas tabelas.

Selecione a tabela desejada para a confecção de sua página.

Selecione os campos que deseja que apareça no formulário ou clique em todos  para selecionar toda a tabela.

Nota importante:
Para o perfeito funcionamento do formulário ASP o primeiro campo a ser selecionado deve obrigatóriamente ser um campo indexado na tabela, o seja, um campo do tipo "AutoNumeração" ou o identificador da tabela para que os registro possam ser manipulados com perfeição.

Manipulando os registros da tabela...


 


Após selecionar os itens da tabela será exibida a tela acima, nela faz-se a formatação dos dados e da apresentação do itens no formulário

Cada item selecionado no campo "Campos do Formulário" serão apresentados a direita em "Configurações dos campos do Formulário" onde os dados serão formatados.
Nele pode-se formatar várias opções de campo como segue:

Parâmetro

Descrição

Nome do rótulo para este campo.

Ex. "DEQUEM" para "DE QUEM"

Este campo aceita valores nulos?

Se estiver marcado não haverá a necessidade preencher o campo. Para torna-ló obrigatório retire o tik.
