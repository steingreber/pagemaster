.::Caminho

Ao iniciar o sistema pela primeira vez deve-se definir onde os arquivos serão gravados juntamente com as imagens usados no mesmo. [Ctrl+M]

Caso não informe o caminho será usado o padrão. mas as imagens das páginas gerada não serão exibidas.

Toda a página criada com o PageMaster devera acompanhar a pasta images que é o lugar onde estas imagens serão gravadas.
